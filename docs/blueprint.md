# **App Name**: WikiSense

## Core Features:

- Wikipedia Search: Search Wikipedia for relevant articles based on user query.
- Genkit Summarization: Utilize Genkit to process and summarize the search results.
- Contextual Understanding: Employ Genkit's reasoning capabilities to decide when to present information for enhanced understanding of Wikipedia results, acting as a tool.
- Display Results: Show the original search results alongside the Genkit-generated summary, also display images if the user search for image.
- User Input: Provide an input field for users to enter their search queries.

## Style Guidelines:

- Primary color: Light, calming blue (#64B5F6) to evoke trust and knowledge.
- Background color: Soft, off-white (#F5F5F5) for readability and minimal distraction.
- Accent color: A slightly darker shade of blue (#42A5F5) to highlight important elements such as the search button or key information in summaries.
- Body and headline font: 'Inter', sans-serif, providing a modern and neutral appearance for both headlines and body text, optimizing readability for search results.
- Use simple, clear icons to represent actions or sources.
- Maintain a clean, organized layout for presenting search results and Genkit summaries side by side or in an alternating format for clarity.
- Subtle transitions and animations to improve the user experience, like loading animations when retrieving and summarizing information.
